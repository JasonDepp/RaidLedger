# RaidLedger

专为为目前金团设计
团长可以记账，方便快捷
打工可以记账，童叟无欺

 * [Wowace](https://www.wowace.com/projects/raidledger)
 * [NGA连接](https://bbs.nga.cn/read.php?tid=18961750)

## 使用
界面呼出
```
/gtuan
```

添加物品
```
/gutan [物品]
```

 * 消费前5名汇总广播
 * 随时修改价格，拾取人
 * 智能自动完成，输入 1 会显示小队1 的成员 输入 猎人 会把 猎人 或者名字带有猎人的人都列出来
 * 可以添加补助，会在总钱数中自动扣除
 * 人数自动计算平均费用
 * 拾取的物品会自动加到账簿 当然还可以手动添加 (物品) 右键点击物品 可以删除记录
 * 可以导出文本 发到 微信群等地方
